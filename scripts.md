### Delete HPA
```sh
oc get hpa -A --no-headers | awk '{print $1}' | sort | uniq > config
while read -r i ; do  for item in `oc get hpa --no-headers -n $i | awk '{print $1}' `; do  oc delete hpa $item -n ${i};     done; done < "config"
    ```

### List DC

### List Pod
